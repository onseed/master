1. 기본: $('.modal_cont').uxLayerPop();

2. 자동 열기: $('.modal_cont').uxLayerPop({auto: true});

3. 자동 닫기: $('.modal_cont').uxLayerPop({popClose: true});

4. 이중 팝업: $('.modal_cont').uxLayerPop({qspop: true});

5. 콜백 처리: $('.modal_cont').uxLayerPop({callback: function(){ //오픈시 콜백처리} });

6. maxW: $('.modal_cont').uxLayerPop({maxW: 800});

7. maxH: $('.modal_cont').uxLayerPop({maxH: 300});

8. scrlLock: $('.modal_cont').uxLayerPop({scrlLock: true, callback: function(){ $(".ui_scrollview").uxScrollView(); //uxScrollView 실행 }});

휠스크롤 막기 옵션(scrlLock)시 window내 모든 스크롤이 binding되어 uxScrollView을 사용해야함.

$(".ui_scrollview").uxScrollView(); //uxScrollView 실행
